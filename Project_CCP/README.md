## Project Overview
This project aims to analyze customer churn behavior using telecom customer data.
The goal is to identify key factors influencing churn and provide business insights.

## Project Steps Completed
- Data Understanding
- Data Cleaning
- Exploratory Data Analysis (EDA)

## Key Insights
- Month-to-month customers have higher churn
- Customers with higher monthly charges churn more
- Low-tenure customers are at higher churn risk
